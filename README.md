# Personal Research Website

A lightweight multi-page academic website for Dr. Mei Ling Soh, designed to deploy cleanly on GitHub Pages.

## Pages

- `index.html` - home page
- `about.html` - biography, training, teaching philosophy, CV download
- `research.html` - research agenda, methods, current questions
- `publications.html` - publications, manuscripts, talks, grants
- `news.html` - recent updates and milestones
- `contact.html` - contact details, skills, language profile

## Shared Assets

- `styles.css` - editorial-inspired layout and responsive styling
- `script.js` - hero slider on the home page and mobile navigation
- `mei-ling-soh-cv.pdf` - downloadable CV

## Publish on GitHub Pages

1. Create a GitHub repository
2. Upload these files to the repository root
3. In GitHub repository settings, enable GitHub Pages from the `main` branch root
4. If the repository is named `yourusername.github.io`, the site will publish at that domain
